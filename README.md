# My Portfolio

I built this for myself — to finally have a portfolio that actually looks the way I wanted, built the way I like to build things (React, TypeScript, Tailwind).

If you stumbled across this and want to use it for your own portfolio, go for it. No strings attached.

---

## What's inside

- **React 18 + TypeScript** — because I can't go back to JS-only anymore
- **Vite + SWC** — fast builds, no complaints
- **Tailwind CSS v4** — config-free, utility-first
- **React Router v7** — single page in practice, but structured properly

---

## Getting Started

```bash
git clone https://github.com/heniums/portfolio.git
cd portfolio
npm install
npm run dev
```

---

## Making it yours

Almost everything you'd want to change is in one file:

```
src/config.ts
```

Open it, swap in your info, done. There's content for your bio, skills, work experience, projects, tech stack, and contact links — all in one place.

Your photo goes in `src/assets/me.jpg`. Nav links are in `src/main.tsx`.

---

## Scripts

```bash
npm run dev       # Dev server
npm run build     # Type-check + build
npm run lint      # ESLint
npm run preview   # Preview production build
```
